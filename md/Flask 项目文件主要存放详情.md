# Flask 项目文件主要存放详情

- #### project/

  - app/			整个程序包含的目录

    - static			静态文件资源
      - js/			Js脚本
      - css/			样式表
      - img/			突变
      - favicon.ico			网站图标

  - templates/			模板文件

    - common/			通用模板
    - errors/			错误页面
    - user/			用户模板
    - posts/			帖子模板
    - email/			邮件发送

  - views/			视图模板
